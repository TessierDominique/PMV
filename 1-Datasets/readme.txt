
Repertoire scripts:
Contient les scripts Python pour ajouter aux enregistrements du fichier des stations Bixi: l'altitude, le code postal et la ville.
Note 1: Il faut un API key Google pour récupérer l'information


Répertoire sources:
- Contient les dataset de la météo pour l'année 2017 (référence: http://climat.meteo.gc.ca/historical_data/search_historic_data_f.html)
- Contient le fichier décrivant les stations Bixi 
- Contient la liste des entrées de métro de la STM (référence: http://www.stm.info/fr/a-propos/developpeurs)

Note: Le data set sur les déplacements en Bixi pour l'année 2017 peuvent être récupéré à: https://www.bixi.com/en/open-data
